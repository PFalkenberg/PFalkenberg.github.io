Video Store Manager

Technologies Used: HTML, CSS, Bootstrap, Javascript, JQuery, AJAX, C#, SQL Server

My video store manager is a full stack web application that I built using ASP.NET, MVC, and Entity Framework. This application allows management over a video rental store. There is a user authentication system in place, with two account types: Admin and Employee. The employee account can view Movies in stock in the store, manage Customers, and create and return rentals. The Admin account can do all the same actions as the Employee account, with the added permissions of being able to add, edit, and remove movies from the store.

Look Based Media Player

Technologies Used: C#, Python, OpenCV

My look based media player application is a program that I worked on with 3 other people. The program is a media player that uses computer vision (camera) to detect whether or not the user is looking at the screen. The media playback will automatically pause when it detects the user not looking and will automatically resume playback when the user looks back at the screen. My contribution to this project was the facial detection aspect of it. I used Python and OpenCV library to detect the users face, and send out a flag that tells whether or not the user is looking at the screen.

Keystroke Biometric Analyzer

Technologies Used: Java, JavaFX, ApachePOI

This program reads raw keystroke biometric data from an excel spreadsheet and uses the data to calculate template vectors, generate scores by using probe vectors against the templates, and then calculates the false positive and false negative rates at each threshold.
